# Textual TODO App

A simple TODO app built with Textual in Python.
Mostly for demonstration and educational purposes.
[See the related tutorial here][tutorial].

![](_static_app_demo.png)


## Installation

Install with pip:

```
python -m pip install textual-todo
```

To run:

```
todo
```
